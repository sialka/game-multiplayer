# Jogo Multiplayer

Neste projeto estudei o funcionamento de um jogo multiplayer onde foi usado Javascript + Node.js + Socket.io.

<img src="img-01.png" alt="img-01" style="zoom:75%;" />

**Start Game**

```bash
$ node server
```
